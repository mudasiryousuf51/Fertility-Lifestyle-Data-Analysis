# 📊 Fertility & Lifestyle Data Analysis

## Project Overview
This project focuses on analyzing a fertility dataset to understand how lifestyle and medical factors influence fertility diagnosis.

The goal is to extract meaningful insights using Exploratory Data Analysis (EDA) and visualization techniques.

##  Dataset Information
The dataset contains 100 records with the following features:

- Season  
- Age  
- Childish diseases  
- Accident or serious trauma  
- Surgical intervention  
- High fevers in the last year  
- Frequency of alcohol consumption  
- Smoking habit  
- Number of hours spent sitting per day  
- Diagnosis (Target Variable)

---

##  Objectives
- Analyze the distribution of fertility diagnosis  
- Study the impact of lifestyle habits (smoking, alcohol, sedentary behavior)  
- Evaluate medical history factors (fever, surgery, trauma)  
- Identify patterns and trends affecting fertility  

##  Tools & Technologies
- Python  
- Pandas  
- Matplotlib  
- Seaborn  

##  Exploratory Data Analysis

The following analyses were performed:

- Diagnosis distribution using pie chart  
- Age comparison across diagnosis using boxplot  
- Smoking and alcohol impact using countplots  
- Medical history impact (fever, surgery, trauma)  
- Seasonal trends in diagnosis  

A combined visualization dashboard was created using multiple plots for better insights.

##  Key Insights

- Lifestyle factors such as smoking and alcohol consumption show variation across diagnosis groups  
- Medical history (fever, surgery, trauma) may influence fertility outcomes  
- Age and sedentary behavior indicate noticeable patterns  
- Seasonal variation suggests possible environmental influence
